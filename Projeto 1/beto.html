Projetos realizados pelo aluno:

O projeto criado para a matéria chamada Modelagem e Simulação tem como finalidade implementar conhecimentos de física e programação para modelar uma situação física em Python. No projeto do qual participei, foi modelado a força G da Centrifuga Tsf-18, uma centrifuga russa utilizada por astronautas para criar resistência à força G. Como resultados obtivemos uma função que retorna a força G pelo tempo e assim pudemos determinar diversas variáveis, como por exemplo, quanto tempo uma pessoa demora para desmaiar ou também morrer, podendo, por exemplo, evitar que isto aconteça.
Esse projeto consistiu em dar uma palestra sobre o tema: “Grandes construções, sonhos e Quimeras” cujo tema teve inicialmente uma abordagem histórica sobre construções e infraestruturas e posteriormente com desenvolvido do tema em torno do Burj Khalifa (maior prédio do mundo, localizado em Dubai) tanto no âmbito técnico, como a construção por exemplo, quanto no âmbito social, econômico e político. Há diversos conceitos tecnológicos para a construção de uma infraestrutura desse porte no deserto e para superar os desafios que sua mega construção impõe, além disso há todo um entendimento do por que a humanidade procura pelo construção de prédios cada vez mais altos.
A estação meteorológica, projeto desenvolvido na matéria chamada Instrumentação e medição, teve como finalidade a construção de um circuito que utiliza um micro controlador chamado Arduino e sensores de temperatura, umidade e pressão, envoltos em uma caixa de livre escolha (criada na máquina a laser). O objetivo desse projeto foi aprender a calibrar os sensores, utilizar softwares para fresamento da PCB (placa para o circuito), implementar conhecimentos de elétrica para os equacionamentos e colocar a mão na massa para aprender a construir o circuito, soldar, etc.
Design de software tinha o intuito de ensinar programação em Python para poder aprender a lógica do código e sua funcionalidade. O projeto desenvolvido é um jogo de Survivor horror chamado Lights Out que foi criado utilizando a linguagem Python e a biblioteca pygame.
Este projeto é constituído por um brinquedo bio-inspirado cujo objetivo era auxiliar o desenvolvimento tanto motor quanto cognitivo de uma criança da faixa etária de 6 a 8 anos. Para isto, primeiramente foi feita uma pesquisa para se direcionar e compreender qual a etapa em que o público alvo está no quesito do desenvolvimento cognitivo e motor. Contendo estas informações foi desenvolvido um projeto nomeado Mazoo, o qual foi criado utilizando madeira (MDF) cortada na máquina a laser e tem como objetivo levar uma bola de um ponto A à um ponto B movendo dois braços os quais alteram inclinação da plataforma cuja bola esta situada.
